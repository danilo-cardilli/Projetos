# Challenge ONE | Java | Back-end | Hotel Alura

##  Sobre o Projeto:
 Projeto: realizado como desafio durante o programa ONE-Alura com o nome de Hotel-Alura. Durante esse desafio foi disponibilizado uma interface com a tecnologia SWING do java, e o participante tinha que realizar a parte back-end. Foi usado o padrões MVC, PreparedStatement e DAO.

## Pré - Requisitos:
- jdk11
- jre11
- eclipse
- windows
- mySql8.0 
- mySql workbench 8.0 C.E


## Instalação:
Esse projeto deve ser instalado os arquivos .jar que estão na pasta 'JARS'.

Clique com o botão direito no projeto e após clique em Properties. Abrindo uma caixa de opções clique em Java Build Path, depois na Libraries,  depois na Modulepath.

Selecione o botão Add JARs…   e adicione todos os .jar.
Dentro do projeto tem um diretório chamado 'bd', onde se encontra o script.sql para a instalação do banco de dados.
